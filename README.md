# Photo Uploader & Gallery

SPA built with ReactJS, NodeJS and MongoDB.

## Features:

- MVC Architecture
- Modular structure, arranged by feature

## Backend:

- Node.js + Express
- MongoDB
- JWT for Auth
- AWS S3 for images storage

## Frontend:

- React.js
- Redux + Thunk
- Material Design
